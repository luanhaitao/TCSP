# 教师数据收集器使用说明（中文版）

访问地址：`/src/collector.html`

这个页面是专门给非技术教师使用的录入工具，核心目标：
- 所有字段中文解释
- 必填项清晰标记
- 自动推荐 ID
- 录入后可一键导出 CSV
- 本地文件一键上传并自动生成链接

## 1. 操作流程（建议）

1. 打开“教师数据收集器”。
2. 先点“读取当前基础数据”。
3. 按顺序录入：社团信息 -> 学员成果 -> 素材绑定。
4. 每填完一条，点击“保存到草稿库”。
5. 最后点击“导出三张 CSV”。
6. 将导出的 CSV 提交给科技组，或按既定流程回填在线表格。

## 2. 自动推荐能力

- 社团 ID 自动推荐：`C001/C002...`
- 成果 ID 自动推荐：`A001/A002...`
- 素材 ID 自动推荐：`M001/M002...`
- 学员化名自动推荐：例如 `小创者07`
- 更新时间自动填充当前时间

## 3. 必填与校验

系统会阻止明显错误数据保存：
- 必填项为空
- 链接不是 `http/https`
- 归属对象未选择

## 4. 本地素材上传（无需网盘）

前提：管理员已在系统中启用统一上传服务（Cloudinary）。

上传封面图：
1. 在“社团信息”里选择本地图片文件。
2. 点击“上传并填入封面链接”。
3. 系统会自动把链接写入 `cover_url`。

上传成果素材：
1. 在“素材绑定”里选择本地图片或视频。
2. 点击“上传并填入素材链接”。
3. 系统会自动填入 `media_url`，并自动识别 `media_type`。

## 5. 草稿库说明

- “草稿库”保存在当前浏览器本地。
- 刷新页面不会丢失。
- 可逐条删除，也可全部清空。
- 草稿仅保存在你当前设备和当前浏览器。

## 6. 导出说明

点击“导出三张 CSV”会下载：
- `club_profile_draft_时间戳.csv`
- `student_artifact_draft_时间戳.csv`
- `media_asset_draft_时间戳.csv`

可用于：
- 交给科技组统一导入
- 或复制内容到在线表格（按同名表结构）

## 7. 常见问题

1. 为什么“所属社团ID”下拉为空？
- 先新增社团并保存，或先读取基础数据。

2. 为什么“归属ID”没有内容？
- `owner_type=artifact` 时，需要先有成果；
- `owner_type=club` 时，需要先有社团。

3. 为什么导出后页面还没更新？
- 导出只是生成文件；展示页面要更新在线表格或数据源后，再点“刷新最新数据”。

4. 为什么上传按钮是灰色的？
- 说明管理员还没启用统一上传服务，请先联系科技组完成 `src/config.js` 配置。
