# 教师数据收集器使用说明（中文版）

访问地址：`/src/collector.html`

## 0. 先登录（新增）

1. 打开页面后先在顶部“身份验证”输入姓名。
2. 仅以下人员可进入：
   - 社团执教教师（姓名需与社团表“执教教师”完全一致）
   - 超级管理员（由管理员在 `src/config.js` 的 `auth.adminNames` 预先配置）
3. 登录成功后，系统会自动加载你权限范围内的数据。
4. 点击“退出登录”可结束当前会话。

这个页面是专门给非技术教师使用的录入工具，核心目标：
- 所有字段中文解释
- 必填项清晰标记
- 自动推荐 ID
- 录入后可一键导出 CSV
- 本地文件一键上传并自动生成链接

## 1. 操作流程（建议）

1. 打开“教师数据收集器”并先完成登录。
2. 先点“读取当前基础数据”。
3. 按顺序录入：社团信息 -> 学员成果 -> 素材绑定。
4. 每填完一条，点击“保存到草稿库”。
5. 点击“保存并发布（推荐）”。
6. 系统发布成功后会自动备份并更新数据源。
7. 若自动发布不可用，再点击“导出三张 CSV”走备用流程。

## 2. 自动推荐能力

- 社团 ID 自动推荐：`C001/C002...`
- 成果 ID 自动推荐：`A001/A002...`
- 素材 ID 自动推荐：`M001/M002...`
- 学员化名自动推荐：例如 `小创者07`
- 更新时间自动填充当前时间

## 3. 学员成果 Excel 一键导入

在“新增学员成果”页面可直接：
1. 点击“下载成果导入模板（Excel）”。
2. 按模板中文字段填写（建议每行 1 个学员成果）。
3. 选择 Excel 文件，点击“一键导入成果（Excel）”。

说明：
- 支持按“成果ID”覆盖更新已有草稿。
- 若成果ID留空，系统会自动生成。
- 跳过项通常是必填缺失或成果类型不合法。

模板字段（中文）：
- 成果ID、学员化名、年级、所属社团、成果名称、成果类型、关键词、我的参与内容、成果简介、一句话收获、成长证据、教师简评

说明：`更新时间`不需要在模板中填写，导入后系统自动生成。
说明：`所属社团`可填写“社团名称”或“社团ID”；若有同名社团，请填写社团ID（如 `C001`）。

## 4. 社团信息 Excel 一键导入

在“新增社团信息”页面可直接：
1. 点击“下载社团导入模板（Excel）”。
2. 按模板中文字段填写。
3. 选择 Excel 文件，点击“一键导入社团（Excel）”。

说明：
- 支持按“社团ID”覆盖更新已有草稿。
- 若社团ID留空，系统会自动生成。
- `展示状态`不填时默认 `active`。
- 模板内含“填写说明”页，列出了推荐的社团类别（展馆类别）选项。

## 5. 必填与校验

系统会阻止明显错误数据保存：
- 必填项为空
- 链接不是 `http/https`
- 归属对象未选择

## 6. 本地素材上传（无需网盘）

前提：管理员已启动局域网发布服务（`node scripts/portal_server.mjs`）。

素材目录一键导入（推荐）：
1. 在“素材绑定”页点击“选择素材根目录”。
2. 目录名必须是：`成果ID_成果名称`（例如 `A001_智能红绿灯`）。
3. 点击“开始一键导入素材目录”。
4. 系统会自动完成：上传 -> 绑定 `owner_id` -> 写入素材草稿库。

导入规则：
- 支持后缀：图片 `jpg/png/gif/webp`，视频 `mp4/mov/webm`，文档 `pdf`。
- 仅从目录名前缀提取成果ID进行绑定。
- 视频同名图片会自动作为缩略图（如 `demo.mp4` 配对 `demo.jpg`）。
- 若目录命名不合规、成果ID不存在或文件类型不支持，预检会直接提示并阻止导入。

上传封面图：
1. 在“社团信息”里选择本地图片文件。
2. 点击“上传并填入封面链接”。
3. 系统会自动把链接写入 `cover_url`。

上传成果素材：
1. 在“素材绑定”里选择本地图片、视频或PDF。
2. 点击“上传并填入素材链接”。
3. 系统会自动填入 `media_url`，并自动识别 `media_type`。

## 7. 草稿库说明

- “草稿库”保存在当前浏览器本地。
- 刷新页面不会丢失。
- 可逐条删除，也可全部清空。
- 草稿仅保存在你当前设备和当前浏览器。

## 8. 导出说明

点击“导出三张 CSV”会下载：
- `club_profile_draft_时间戳.csv`
- `student_artifact_draft_时间戳.csv`
- `media_asset_draft_时间戳.csv`

可用于：
- 交给科技组统一导入
- 或复制内容到在线表格（按同名表结构）

## 9. 常见问题

1. 为什么“所属社团”下拉为空？
- 先新增社团并保存，或先读取基础数据。

2. 为什么“归属ID”没有内容？
- `owner_type=artifact` 时，需要先有成果；
- `owner_type=club` 时，需要先有社团。

3. 为什么导出后页面还没更新？
- 导出只是生成文件；展示页面要更新在线表格或数据源后，再点“刷新最新数据”。

4. 为什么上传按钮是灰色的？
- 说明管理员还没启用统一上传服务，请先联系科技组完成 `src/config.js` 配置。

5. 为什么“保存并发布”不可用或失败？
- 先确认管理员是否使用 `node scripts/portal_server.mjs` 启动服务。
- 再确认发布接口地址是否可访问：`http://localhost:8090/api/health`。

6. 为什么“素材目录一键导入”提示预检失败？
- 常见原因：目录名不是 `成果ID_成果名称`、成果ID不存在、文件后缀不受支持。
- 先按页面“命名规则卡片”修正后再导入。
