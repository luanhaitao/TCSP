# 教师自维护说明（即时发布）

## 日常更新流程
1. 打开在线表格（社团表 / 学员成果表 / 素材表）。
2. 新增或修改数据，保存后无需找技术人员。
3. 打开云展页面并点击“刷新最新数据”。
4. 检查是否展示正确（社团页、成果墙、详情弹窗）。

## 三个高频操作
- 新增成果卡：在 `student_artifact` 新增一行，并在 `media_asset` 补一条图片或视频。
- 修改文字信息：直接改对应字段（如简介、收获、教师评语）。
- 替换素材：在 `media_asset.url` 改为新链接。

## 错误处理
- 若页面提示“数据问题”，先检查必填字段是否为空。
- 若图片不显示，检查 `url` 是否是可公开访问的 `http/https` 链接。
- 若视频按钮不出现，说明视频链接无效或未填写。

## 科技组抽检建议
- 每周一次抽检：随机查看 3 个社团、10 条成果卡。
- 关注项：隐私口径（化名）、链接可访问、成果类型是否规范。
